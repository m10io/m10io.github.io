Home

The DRM Platform was designed to manage digital assets such as regulated stablecoins, tokenized deposits, and central bank digital currencies (CBDC). It is optimized for high-volume, low-latency use cases. On this site you will find details on how the core component — the ledger — functions and how you can interact with it through the API using SDKs and a CLI.

The current SDKs available are for Typescript, Flutter and Rust.
